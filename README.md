### 1. **GET /users**  
Доступен без токена. Возвращает список пользователей.  

### 2. **POST /auth/register**  
Регистрация нового пользователя.  

### 3. **POST /auth/login**  
Авторизация. Возвращает токен доступа.  

### 4. **GET /animals**  
### 5. **POST /animals**  
### 6. **PUT /animals/:id** 
### 7. **DELETE /animals/:id** 
Доступны **только с токеном**, полученным при логине/регистрации.
